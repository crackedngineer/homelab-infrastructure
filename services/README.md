# Reconya

Reload systemd
sudo systemctl daemon-reload

3) Start / Stop / Enable Reconya

Start service:
```
sudo systemctl start reconya
```

Stop service:
```
sudo systemctl stop reconya
```

Restart:

```
sudo systemctl restart reconya
```

Enable at boot:
```
sudo systemctl enable reconya
```

Check status:
```
sudo systemctl status reconya
```
